 [Windows] Kodi XBMC - Texture Tool (.xbt)

How to decompile:

1. Select Source File.
2. Select Destination Dir
3. Press Start
4. Wait until done.

How to compile:

1. Select source
2. Select Destination
3. Press Start
4. Wait until done
